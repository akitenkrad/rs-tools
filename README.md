# rs-tools

```text
Usage: tools <COMMAND>

Commands:
  wordcloud  Generate a word cloud from a text file and images
  help       Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```

## WordCloud

```text
Usage: tools wordcloud [OPTIONS] --text <TEXT>

Options:
  -t, --text <TEXT>       Path to the text file
  -i, --images <IMAGES>   Path to the images directory
  -f, --font <FONT>       Path to the font file
  -t, --tokenize          Whether to tokenize the text
  -r, --random-font-size  Whether to use random font size
  -h, --help              Print help
```
